# STRIVE Independence Left-Right Discrimination

A static browser-based hand laterality task for identifying left and right hands.

## Files

- `index.html` - app entry point
- `styles.css` - visual styling
- `app.js` - task logic and scoring
- `Left/` - left hand images
- `Right/` - right hand images
- `logo.png` - STRIVE Independence logo

## Run Locally

Open `index.html` in a browser, or serve the folder with:

```bash
python3 -m http.server 8765
```

Then open:

```text
http://127.0.0.1:8765/
```

## GitHub Upload

Upload the contents of this folder to a GitHub repository. The images in this package have been optimized for web upload.
